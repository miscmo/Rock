详细描述

template<class T, class FromStr = LexicalCast<std::string, T>, class ToStr = LexicalCast<T, std::string>>
class rock::ConfigVar< T, FromStr, ToStr >

配置参数模板子类,保存对应类型的参数值

T 参数的具体类型 FromStr 从std::string转换成T类型的仿函数 ToStr 从T转换成std::string的仿函数 std::string 为YAML格式的字符串
